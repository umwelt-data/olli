# Dialogs

Olli includes several dialogs that give you additional ways to explore and interact with the data. Each dialog opens with a keyboard shortcut and closes with **Escape**, which returns focus to where you were in the tree.

## Help (`?`)

Press `?` to open the help dialog. It shows:

- **Screen reader setup** reminders for your operating system (VoiceOver, NVDA, or JAWS).
- **Keyboard shortcuts** grouped by category — Navigation, Jump to section, and Dialogs. The shortcuts listed are specific to the current tree, so a chart tree shows shortcuts that a diagram tree may not.

Press **Escape** to close.

## Table (`t`)

Press `t` to open the table dialog. It shows the data under your current focus as a table with rows and columns.

- If you're focused on an axis category, you see the data points in that category.
- If you're focused on a higher-level item, you see all the data it contains.

Arrow through the rows and columns to explore the values. Press **Escape** to close and return to the tree.

The table dialog is available in chart trees.

## Filter menu (`f`)

Press `f` to open the filter menu. It lets you add or remove filters on any data field.

- Select a field and choose which values to include or exclude.
- When you close the dialog, the tree rebuilds to show only the data that matches your filters.
- Open the filter menu again to adjust or remove filters.

Press **Escape** to close and return to the tree.

The filter menu is available in chart trees.

## Targeted navigation (`r`)

Press `r` to open the targeted navigation dialog. Type a value to jump directly to the first item in the tree that matches.

This is useful for large trees where pressing **Right Arrow** repeatedly to find a specific category or data point would be slow. Type the value you're looking for and Olli takes you straight there.

Press **Escape** to close and return to the tree.

Targeted navigation is available in chart trees.

## Description settings (`d`)

Press `d` to open the description settings dialog. It lets you control what information your screen reader announces at each item. See [Customizing Descriptions](/using/descriptions) for a full guide to using this dialog.

Description settings are available in all trees.

## General tips

- All dialogs close with **Escape**.
- Focus returns to the tree item you were on before opening the dialog.
- The help dialog (`?`) always lists which dialogs are available in the current tree.
