# Keyboard Controls

This page lists every keyboard shortcut available in Olli. Press `?` at any time inside a tree to open the help dialog, which shows the shortcuts available for the current tree.

## Navigation

These shortcuts are always available when the tree has focus.

| Key | Action |
| --- | --- |
| Up Arrow | Move up one level |
| Down Arrow / Enter / Space | Move down one level |
| Left Arrow | Previous item at this level |
| Right Arrow | Next item at this level |
| Home | First item at this level |
| End | Last item at this level |
| Escape | Move up one level |
| `o` | Go to the top of the tree |

**Returning to the tree:** Press `o` from anywhere on the page (outside a text input) to jump back into the tree. Focus returns to the last item you were on.

## Jump to section

These shortcuts are available in chart trees (not diagrams). They jump directly to a part of the chart.

| Key | Action |
| --- | --- |
| `x` | Jump to the x-axis |
| `y` | Jump to the y-axis |
| `l` | Jump to the legend |

If the chart doesn't have a legend, pressing `l` does nothing.

## Dialogs

These shortcuts open a dialog over the tree. Press **Escape** to close any dialog and return to where you were.

| Key | Action |
| --- | --- |
| `?` | Open help |
| `t` | Open table |
| `f` | Open filter menu |
| `r` | Open targeted navigation |
| `d` | Open description settings |

The table, filter, and targeted navigation dialogs are available in chart trees. The help and description settings dialogs are available in all trees.
